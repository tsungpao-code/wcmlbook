Quadriga_channel_generator.mlx is a MATLAB script that generates the Rayleigh Channel Dataset via 'Quadriga 2.6'.

Please add the Quadriga path to your matlab via:
>>addpath(genpath('D:\MATLAB\Libs\QuaDriGa_2021.07.12_v2.6.1-0'));  % change to your actual path
>>savepath

Exercise_2.4_starter.py is a python script which applies the generated .mat dataset to GAN for channel generation, with a key function left for the reader to realize.
Exercise_2.4_solution.py is the completed script for reference.
The tensorflow version should be 2.x to run the scripts.
